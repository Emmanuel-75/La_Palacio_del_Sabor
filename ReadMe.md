# Simple responsive webpage

A challenge designed to test practical understanding of CSS layouts—especially Flexbox— and Tailwind CSS by building a responsive navbar that switches between a horizontal menu on desktop and  a hamburger dropdown on mobile.

## Features
- Semantic HTML5 structure
- Clean, readable indentation
- Switched from vanilla CSS to Tailwind CSS
- Responsiveness

## Tech Stack
- HTML5
- Tailwind CSS

## Purpose
Practical application of CSS layouts and Tailwind CSS to create a responsive layout


## How to Use
Download the file and run `npx live-server` in the terminal
or
Tap on the live link associated with this folder
